# keleustēs

> A ship's officer in charge of equipment and the crew.

Keleustēs manages the lifecycle of virtual machines on xhyve.

## Dependencies

    $ go get github.com/golang/dep/cmd/dep
    $ go get github.com/mitchellh/gox

## Building

    $ make build
    $ tree .build/

## Testing

    $ make test
